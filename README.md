## Compatibility

This program was compiled with GCC 12.1.0 in Linux Kernel 5.18.

## Building

```sh
make serv
make cli
```
(so hard, yey)

## Runing a server

```
./server2 IP PORT
```
## Running a client
```
./clirc
```
